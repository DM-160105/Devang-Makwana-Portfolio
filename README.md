## ğŸ“– About The Project

This is a **high-performance, interactive portfolio website** designed to showcase my skills, projects, and professional journey. It leverages the latest web technologies to deliver a premium user experience, featuring:

- **Fluid Animations**: Powered by **Framer Motion** and **GSAP** for buttery smooth page transitions and scroll effects.
- **Lazy Loading**: Optimized performance with components like `LazyFooter` that only load when needed.
- **Dark Mode**: Instant, hydration-mismatch-free theme switching.
- **SEO Optimized**: Fully configured `sitemap.xml`, `robots.txt`, and Open Graph metadata for maximum discoverability.
- **PWA Ready**: Installable on mobile devices with a custom Web Manifest.

## ğŸš€ Getting Started

To run this project locally on your machine:

1.  **Clone the repository**

    ```bash
    git clone https://github.com/DM-160105/Devang-Makwana-Portfolio.git
    cd Devang-Makwana-Portfolio
    ```

2.  **Install Dependencies**

    ```bash
    npm install
    # or
    bun install
    ```

3.  **Run Development Server**

    ```bash
    npm run dev
    # or
    bun run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) in your browser.
